## 办公系统
### 系统模块
- 系统管理

### 技术汇总
- springboot
- JPA/Hiberante
- mybatis
- freemarker 模板引擎
- bootstrap 前端模板

### 表结构
- aoa_type_lis(系统类型列表)
- aoa_status_list(系统状态列表)
- aoa_plan_list(系统计划表) 保存了类型列表 id和状态列表 id
- aoa_attachment_list 附件表
- aoa_file_list 文件表
- aoa_file_path 文件路径表
- aoa_user(用户表) 与计划表的关系为一对多
